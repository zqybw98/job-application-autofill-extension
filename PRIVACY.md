# Privacy Policy

## Local Data Only

Job Application Autofill stores profile and application tracking data only in `chrome.storage.local`.

## No External Servers

The extension does not upload, sync, transmit, or sell user data. It does not use external servers.

## No Analytics

The extension does not include analytics, telemetry, tracking pixels, or behavioral monitoring.

## No Gmail or Drive Access

The extension does not request or use Gmail, Google Drive, OpenAI APIs, or any third-party service.

## No Automatic Submission

The extension never clicks submit buttons and never submits job applications. Users must review every page and submit manually.

## File Uploads

The extension does not set file input values programmatically. CVs, cover letters, certificates, and other files must be uploaded manually by the user.

## Clearing Data

Users can remove extension data from Chrome extension storage or by uninstalling the extension.
